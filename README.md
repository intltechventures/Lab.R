## Useful R Commands
- To install a package in R, start the interactive shell, and enter the following command:  
  + ```install.packages("{package name}")```
    * example: ```install.packages("ggplot2")```


- To Update R:
  + ```install.packages("installr")```
  + ```installr::updateR()```
